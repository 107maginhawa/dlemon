**SVG overlay placement (per D6):**
Inside the existing `<div style={{ filter: ..., flex: 1 }}>`, add `position: 'relative'` to that div's style. After the `<canvas>` element, add:
```tsx
<svg
  ref={svgRef}
  className="absolute inset-0 w-full h-full"
  style={{ cursor: toolMode && toolMode !== 'none' ? 'crosshair' : 'default', pointerEvents: toolMode !== 'none' ? 'auto' : 'none' }}
  onClick={handleSvgClick}
>
  {/* Render persisted measurements from useMeasurements */}
  {measurements.map(m => <MeasurementShape key={m.id} annotation={m} pixelSpacingMm={pixelSpacingMm} onDelete={deleteMeasurement.mutate} />)}
  {/* Render in-progress drawPoints as ghost lines/shapes */}
  <DrawingPreview toolMode={toolMode} points={drawPoints} />
</svg>
```

**handleSvgClick:** Converts SVG-coordinate click to canvas-image coordinates accounting for `scaleRef` and `offsetRef`. Appends to `drawPoints`. When shape is complete (2 points for distance, 3 for angle, 3+ for area on double-click):
- For calibration: compute pixelDistance = Euclidean distance of the 2 points, set calibrationPixelDist, open CalibrationDialog
- For distance/angle/area: compute measurementValue (using pixelSpacingMm if available, else raw px), call `createMeasurement.mutate({ type, geometry, measurementValue, measurementUnit })`, reset drawPoints

**MeasurementShape (inline sub-component or same file):**
Renders based on annotation.type:
- 'line' (distance): `<line>` + `<text>` with value + unit near midpoint + small `<circle r="4">` delete handle
- 'angle': `<polyline>` (3 points) + `<text>` with degrees + delete handle
- 'area': `<polygon>` with fill-opacity=0.2 + `<text>` + delete handle

**DrawingPreview (inline sub-component):**
Renders ghost/in-progress shape from drawPoints with dashed stroke.

**CalibrationDialog integration:**
Render `<CalibrationDialog open={calibrationOpen} pixelDistance={calibrationPixelDist} onConfirm={handleCalibrationConfirm} onCancel={() => setCalibrationOpen(false)} />` at end of component. `handleCalibrationConfirm`: calls PATCH /dental/imaging/images/{imageId}/calibration with `{ pixelSpacingMm: actualMm / calibrationPixelDist }`, calls onCalibrationSaved prop, resets drawPoints.

**useMeasurements integration:**
Call `const { measurements, createMeasurement, deleteMeasurement } = useMeasurements(imageId)` at top of component.

Keep all existing pan/zoom/rotate/flip/brightness/contrast functionality intact. SVG pointer events are disabled (`pointerEvents: 'none'`) when toolMode is 'none', so pan continues to work via canvas mouse handlers when no tool is active.

### Wave 2: PatientImageList checkbox selection

**Task 2.1** — Modify `apps/dentalemon/src/features/imaging/components/patient-image-list.tsx`

Changes:
1. Add `onCompare?: (items: [PatientImageItem, PatientImageItem]) => void` to `PatientImageListProps` interface
2. Add state inside the component function:
   ```typescript
   const [selectedIds, setSelectedIds] = useState<Set<string>>(new Set())
   ```
3. Add toggle handler:
   ```typescript
   function toggleSelect(item: PatientImageItem, e: React.ChangeEvent<HTMLInputElement>) {
     e.stopPropagation()
     setSelectedIds(prev => {
       const next = new Set(prev)
       if (next.has(item.id)) {
         next.delete(item.id)
       } else if (next.size < 2) {
         next.add(item.id)
       }
       return next
     })
   }
   ```
4. Add "Compare ▶" button in header — visible only when `selectedIds.size === 2`:
   ```tsx
   {selectedIds.size === 2 && (
     <button
       onClick={() => {
         const selected = (data?.items ?? []).filter(i => selectedIds.has(i.id))
         if (selected.length === 2) {
           onCompare?.([selected[0]!, selected[1]!])
         }
       }}
       className="bg-[#FFE97D] text-black text-xs font-semibold px-3 py-1.5 rounded-md"
       data-testid="compare-btn"
     >
       Compare ▶
     </button>
   )}
   ```
5. Restructure each `<li>` to separate checkbox `onChange` (stops propagation) from content `onClick`:
   - `<li key={item.id} className="flex items-center gap-3 p-3 hover:bg-zinc-50">`
   - `<input type="checkbox" checked={selectedIds.has(item.id)} onChange={(e) => toggleSelect(item, e)} className="shrink-0 accent-[#FFE97D]" data-testid={`select-image-${item.id}`} />`
   - Wrap existing content in `<div className="flex-1 min-w-0 cursor-pointer" onClick={() => onSelectImage?.(item)}>`
   - Remove any `onClick` from the `<li>` itself

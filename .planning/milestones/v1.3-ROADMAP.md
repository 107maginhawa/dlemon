# Milestone v1.3: Imaging Workspace

**Status:** Complete ✅
**Shipped:** 2026-05-11

## Overview

Full dental imaging workspace: upload X-rays, canvas viewer with zoom/pan/rotate/brightness, measurements (distance/angle/area) with calibration, annotations (label/arrow/freehand/shape/tooth), side-by-side comparison, and offline caching via Tauri FS + IndexedDB. 18 IMG requirements shipped.

## Milestone Summary

### Key Decisions (D1–D10 from eng review)
- D1: Chunked upload prerequisite in Phase 1.5
- D2: imagingTier stub column (free/basic/addon default free)
- D3: Tauri FS for offline blob storage
- D4: Phase 3 split into 3a (P0) + 3b (P1) to unblock comparison
- D5: Union adapter (imaging_study + legacy dental_attachment) via source discriminator
- D6: Zod-validated geometry JSONB (not typed columns)
- D7: Scripted requestAnimationFrame benchmark (500 frames)
- D8: Source discriminator 'imaging' | 'legacy' on PatientImageItem
- D9: Calibration persists immediately (not session-local)
- D10: Modality non-nullable with default 'other'

Flow Summary

Step 1: Condition
  • SVG surfacemap — click zones to select surfaces
  • Selected surfaces shown as pills below
  • Pick condition from 6-option grid
  • Surfaces + condition required before Next
Step 2: Treatment
  • Summary chip shows selected condition
  • Treatment plan name (text input)
  • CDT code (optional)
  • Notes (optional textarea)
  • Estimated cost (number input)
Step 3: Review
  • Full summary of all fields
  • List of existing conditions on this tooth
  • Save applies per-surface coloring to chart
  • Adds row to breakdown table

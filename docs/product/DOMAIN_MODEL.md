         check-in(BR-001)    dentist complete    scheduled job

    │                    │
    │  (auto-discard)     │  (auto-discard)
    └────────────────────►│
                          └──► discarded  [BR-005 — deferred, ADR-010]
```

| From | To | Actor | Condition | Event Published |

| active | completed | Dentist | — | DE-002 |

| active | discarded | System | BR-005 (not enforced) | — |

### SM-TREATMENT: Treatment Lifecycle

```
diagnosed ──► planned ──► performed ──► verified
    │             │            │
    └─────────────┴────────────┴──► dismissed
```

| From | To | Actor | Condition | Event Published |
|------|----|-------|-----------|----------------|
| diagnosed | planned | Dentist | Visit active | — |
| planned | performed | Dentist | Visit active (BR-006) | DE-005 |
| performed | verified | Dentist-Owner | — | — |

```
